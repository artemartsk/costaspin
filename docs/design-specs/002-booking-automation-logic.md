# 🧠 Business Logic: Post-Booking Automation (Epic 2)

**Status:** ACTIVE  
**Feature:** Сценарии взаимодействия (WhatsApp) после ручного создания записи в календаре.

## Фундамент Системы (Core Rule)
В клинике **CostaSpine Elviria** основным каналом коммуникации платформы является WhatsApp. Искусственный интеллект (VAPI/Twilio) и автоматические формы (Onboarding) опираются строго на номер телефона. 
**Следствие:** При создании нового пациента поле `Phone Number` является **СТРОГО ОБЯЗАТЕЛЬНЫМ (Required)**. Без телефона цепочка автоматизации обрывается.

---

## 🌟 Флоу 1: Ручная запись Нового Пациента
**Триггер:** Администратор нажимает `[Save Booking]` на вкладке *Create New Patient*.

1. **Запись в БД (Patients):** Cоздается профиль, `forms_completed = false`. Пациент получает уникальный `id`.
2. **Запись в БД (Appointments):** Cоздается запись с меткой `booking_source = 'manual'`.
3. **Блокировка слота:** Возникает мгновенный конфликт-лок для AI Voice Agent (робот перестает предлагать этот слот).
4. **Trigger: WhatsApp Onboarding:** Инфраструктура ловит создание записи и отправляет приветственный текст на обязательный номер телефона:  
   > *"Добро пожаловать в CostaSpine Elviria! Ваш прием подтвержден на {Date} в {Time}. Пожалуйста, заполните медицинскую анкету до визита по этой ссылке: [Magic Link]"*
5. Чекпоинт: Пациент заполняет анкету -> `forms_completed` меняется на `true`.

---

## 🔄 Флоу 2: Ручная запись Существующего Пациента
**Триггер:** Администратор загружает профиль на вкладке *Existing Patient* и жмет `[Save Booking]`.

1. **Прямое бронирование:** В БД резервируется слот `booking_source = 'manual'`.
2. **Проверка статуса анкеты:** 
   - Если `forms_completed == true` (то есть пациент уже заполнял анкеты ранее).
   - Если `forms_completed == false` (старый пациент из Гуадальмины, у которого еще нет цифровой карты).
3. **Trigger: WhatsApp Notification:**
   - Если анкета заполнена: Легкое напоминание:  
     > *"Здравствуйте, {Name}! Ждем вас на повторный прием {Date} в {Time}".*
   - Если анкета НЕ заполнена: ИИ классифицирует его как "Нового для системы Elviria" и запускает цепочку из Флоу 1 (с отправкой Magic Link на заполнение формы).
