# 🎨 Design Spec: Communications Timeline (EMR)

**Reference:** TASK 14: AI Communications Timeline
**Status:** ACTIVE

## ✨ UX/UI Концепция
Нам нужно объединить разные каналы коммуникации (Звонки и WhatsApp-сообщения) в единую ленту, избегая визуального шума таблиц. Идеальный референс: история банковских транзакций или лента активности в GitHub / Notion.

## 📐 Текстовый прототип (Wireframe)

```text
============================================================
  Communications History
============================================================

  [ 📅 16 April 2026 ]

  |
 (📞)  AI Phone Call (Inbound) • 14:32 (3 min)
  |    [ Status: Completed ] 
  |    +--------------------------------------------------+
  |    | ▶ 0:00 ━━━━━━●────────────── 3:00   < audio >    |
  |    +--------------------------------------------------+
  |    [ ▾ Show Transcript ]                              | <-- Accordion
  |      [Bot]: Hello! This is CostaSpine Elviria...      |
  |      [Patient]: Hi, my back hurts really bad!         |
  |
  |
 (💬)  WhatsApp Thread • 10:15
  |    [ Status: Active ]
  |    +--------------------------------------------------+
  |    | [ CostaSpine AI ]                                |
  |    | Hi, confirming your appt for tmrw...             |
  |    |                                                  |

  |    +--------------------------------------------------+

  [ 📅 14 April 2026 ]

  |
 (📞)  AI Phone Call (Outbound) • 09:00
  |    [ Status: No Answer (Red Badge) ]

```

## 🛠 Задачи для фронтенд-разработчика

1. **Вертикальная направляющая (Timeline Bar):** 
   - Сделайте в верстке постоянную вертикальную линию слева (например, `border-l-2 border-slate-200 ml-4`), чтобы визуально "нанизать" все события.
   - Иконки (📞, 💬) нужно позиционировать абсолютно (поверх этой линии) в круглых белых бейджах.
   
2. **Дизайн чатов (Chat Bubbles):**
   - Для диалогов из `whatsapp_threads` не используйте простой текст. Отрендерьте их в виде "бабблов", как в мессенджере.
   - Если `role === 'assistant'` — серый пузырь слева.
   - Если `role === 'user'` (пациент) — светло-зеленый (`bg-emerald-50` или `bg-blue-50`) пузырь справа. Это моментально считывается.

3. **Скрытые транскрипции (Accordion):**
   - Транскрипции звонков бывают огромными. Они **обязаны** быть скрыты под компонентом `Accordion` от `shadcn/ui`, чтобы не "порвать" интерфейс админки бесконечной простыней текста.

4. **Музыкальный плеер:**
   - Если есть ссылка на запись звонка (`recording_url`), рендерите стандартный аудио-элемент `<audio controls className="w-full h-8" src={recording_url}>`.

Эти дизайн-требования дополняют таску менеджера (TASK 14) и создают по-настоящему премиальный UI.
